Combinando Frames

Podemos também colocar um frame dentro do outro e assim combina-los. Por Exemplo aqui, temos três frames em horizontais e dentro do segundo frame horizontal temos três outros frames verticais.

O código para essa página é: 

                < frameset rows="10%,*,10%">
                    < frame src="link">
                    < frameset cols="20%,*,20%">
                        < frame src="link">
                        < frame src="link">
                        < frame src="link">
                    < /frameset>
                    < frame src="link">
                < /frameset>
