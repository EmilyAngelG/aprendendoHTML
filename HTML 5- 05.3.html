Dentro da primeira linha do <FRAMESET> podemos declarar COLS ou ROWS que receberão os valores de cada aba correspondentes ao tamanho dela em pixels, porcentagem ou * que Ã© o alinhamento padrão. COLS é utilizado para alinhamentos verticais e ROWS para alinhamentos horizontais.

Esse código por exemplo tem a seguinte estrutura: 

        < frameset cols="*,*,*,*">
            < frame src="link 1">
            < frame src="link 2">
            < frame src="link 3">
            < frame src="link 1">
        < /frameset>
